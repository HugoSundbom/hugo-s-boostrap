Hugos lexikon över allt om Webben

Element

Ett element är en väldigt grundläggande dela av någon hemsida. Typen av element har en påverkan på hur webbläsaren kommer att tolka innehållet i den. Exempel på element är p och img

HTML

HTML står för HyperText Markup Language och är det språket som alltid används i någon form på hemsidor. Det används för strukturen på hemsidan.

ID

Om du lägger ett ID på ett element så kommer det att vara unikt till det elementet och detta låter dig referare till den delen av hemisidan genom att länka direkt dit eller modifiera elementet med js.

Class

En class används av webbläsaren för att koppla samman specifik css kod med ett html element. Används ofta för design och kan även integreras med javascript.

Meta

Meta är ett ord från grekiskan som betyder mellan, efter eller bakom. Exempel på hur det kan användas är "metadata" som då är data om data.

Responsive

Responsive ett sätt att utveckla hemsidor med dynamiska förändringar baserat på skärmen det visas på. Exempelvis hur menyer lägger sig om skärmen blir bredare eller smalare.

Http

Http är ett protokoll som används för att överföra hemsidor från servrar. De filer som förs över är oftast html, css, js och bilder.

Https

Https är en utveckling på Http protokollet med skillnaden att https är krypterad vilket gör den till det säkrare alternativet och det som är populärast idag.

DNS

DNS är en förkortning på "Domain Name System" och detta system gör det enklare att adressera på exempelvis internet. Systemet kopplar samman IP-adresser och domännamn.

CDN

CDN står för "content delevery network" och är server placerade på olika geografiska platser med syftet att göra så att man kan få tillgång till servern snabt. Den håller alltså en kopia på en hemsida.

Compressed data

Compressed data är data som på något sätt minskas i storlek utan att tappa funktionalitet. Vid compression av bilder kan exempelvis "artifacts" från comrpessionen synas då reultat ej alltid blir perfekt.

Local Bootstrap

Local Bootstrap är när du laddar ner Bootstrap lokalt så att du kan göra en lokal länk till den i html documentet. Om det inte är lokalt går det ändå att länka till, då får du även de senaste uppdateringarna av det.

JQuery

JQuery är det mest populära javascript biblioteket, till stor anledning eftersom det är väldigt lätt att jobba med och

Flexbox

Är en webblayoutmodell som är baserad på 2 axlar som låter dig placera saker relativt till varandra på dem axlarna, samt i början, mitten och slutet av varje axel.

Grid

Är en webblayoutmodell som delar in hemsidan i olika rutor likt ett excel ark. Du kan sedan bestämma inom vilken ruta du vill ha varje element.

Code snippets

code snippets är en bit kod som går att återanvända vilket gör kodandet mindre repetetivt.

WASM

Web assembly är kod skriven i ett av ett flertal språk såsom c++ eller c#. Genom att "compilea" till web assemby så kan man köra det i en webläsare utan att tappa så mycket prestanda.

Chromium

Chromium är grunden för ett flertal av dem större webläsarna, bland annat google chrome och microsoft edge. Chromium är även open source

svg

svg är ett filformatför bilder. Detta format stöds av alla stora webbläsare. Den mest positiva med detta formatet är att det är skalbart vilket innebär av oavsett hur stor du gör bilden så kommer det att funka.

png24

PNG24 är ett bildformat som som har 24 bitars färginformation, formatet behåller också mycket detaljer i bilden.
